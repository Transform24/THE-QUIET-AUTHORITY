## What Each Platform Does

### Google Drive
- Store: raw video files, scripts, devotional drafts, images
- Agent reads from here to repurpose content
- Folder structure to create:
  ```
  /Sanctuary Grace/
    /content-inbox/      ← drop new content here (agents read this)
    /content-queue/      ← repurposed copy waiting to post
    /pin-images/         ← Canva exports for Pinterest
    /email-drafts/       ← MailerLite email drafts
    /leads/              ← lead log backup
  ```

### Pinterest
- YOUR #1 discovery platform for this audience
- Post: 1–3 pins per day (batched weekly)
- All pins link to: assessment site or Beacons
- Boards to create:
  - Sacred Space & Stillness
  - Christian Women Growth
  - Bible Study & Devotionals
  - Faith-Based Wellness
  - Sanctuary Grace Ministry

### MailerLite (email engine)
- Free up to 1,000 subscribers
- Automations: welcome sequences per Silence Profile
- Broadcasts: weekly devotional or ministry update
- DO NOT use Substack for email — MailerLite owns this

### Beacons
- Link-in-bio for Pinterest, YouTube, social
- Storefront for free + paid resources
- Keep updated with current products (Agent 03 audits this)
- URL: https://beacons.ai/sanctuarygrace

### Substack (optional)
- Use ONLY as a public blog for SEO
- Repurpose agent output goes here as long-form posts
- Do NOT send emails from here — use MailerLite
- Free forever for basic use

## Integration Map

```
YouTube video uploaded
    → drop transcript in Google Drive /content-inbox/
    → run Repurpose Agent (Claude trigger)
    → output: pin copy + email + captions → /content-queue/
    → post pins to Pinterest (manual or Pin Agent)
    → send email via MailerLite broadcast

Someone finds pin on Pinterest
    → clicks to assessment site
    → takes 8-question assessment
    → submits name + email (Formspree)
    → Zapier fires → adds to MailerLite
    → MailerLite welcome sequence starts
    → Day 1: welcome + profile
    → Day 3: devotional
    → Day 7: Beacons storefront CTA

They buy from Beacons / Stripe
    → revenue
    → you minister + get paid
```
