### TIKTOK — OFF-CAMERA SETUP

Account: Create @thequietauthority (separate from Instagram)

**3 formats. Zero face required.**

#### Format 1 — Voice + Static Image (start here)
- Record voice memo on phone: read one scripture or 30-second devotion, slowly
- In TikTok: upload static image (Pinterest Canva design) + your voice memo as audio
- Add text overlay with the key line
- Length: 15–30 seconds

#### Format 2 — Text Slideshow (zero voice needed)
- TikTok photo slideshow feature: 5–7 slides, one line each
- Black background (#0d0d0d) — matches TQA brand
- White or gold text — centered
- Trending audio underneath (TikTok suggests it)
- Source: take any profile description, one sentence per slide

#### Format 3 — Text Over B-Roll (most reach)
- Use TikTok's free B-roll (candles, morning light, still water — in the app)
- Large centered text overlay, one line, white on black
- Trending calm/worship-adjacent audio
- 15–30 seconds

**Posting rhythm:** 3x per week — Mon / Wed / Fri
**Source:** Use Pinterest caption text reformatted as 5-slide text cards. One caption = one TikTok.
